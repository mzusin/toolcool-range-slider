Responsive range slider with accessibility support

Mobile-ready range slider that looks good on any devices. It accessible via ARIA-attributes, and supports touch, mousewheel events, and keyboard.

ES6 & Typescript

The library is built with Typescript and does not use external dependencies. The core is 27KB minified or 9KB compressed (GZip).
